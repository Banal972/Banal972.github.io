프로그래머스 - 완주하지 못한 선수

코딩 테스트

2024-08-09

4분

문제

수많은 마라톤 선수들이 마라톤에 참여하였습니다. 단 한 명의 선수를 제외하고는 모든 선수가 마라톤을 완주하였습니다.

마라톤에 참여한 선수들의 이름이 담긴 배열 participant와 완주한 선수들의 이름이 담긴 배열 completion이 주어질 때, 완주하지 못한 선수의 이름을 return 하도록 solution 함수를 작성해주세요.

제한사항

  • 마라톤 경기에 참여한 선수의 수는 1명 이상 100,000명 이하입니다.
  • completion의 길이는 participant의 길이보다 1 작습니다.
  • 참가자의 이름은 1개 이상 20개 이하의 알파벳 소문자로 이루어져 있습니다.
  • 참가자 중에는 동명이인이 있을 수 있습니다.

풀이과정

처음에는 forEach을 사용해서 문제를 풀었었습니다.

이렇게 했을경우 해당 문제에 대한 답을 얻을수 있었지만
효율성 테스트에서는 실패 하였습니다.

효율성에 테스트 하지못한 이유는 시간 복잡도와 관련이 있었습니다.

forEach는 O(n) 이고
indexOf 또한 O(n),
splice 또한 O(n) 입니다.

결국 전체 시간 복잡도는 O(n) * O(n) 이기 때문에 O(n^2)가 됩니다.

그렇기 때문에 효율성을 통과하지 못하게 되었습니다.

효율성을 통과하기 위해서 Map을 하용하게 되었고 해당 방법으로 효율성을 통과할 수 있게 되었습니다.

이 풀이 방법은 participant를 순회하면서 각 참가자의 이름을 가져옵니다.
가져온 참가자의 이름을 map.set()을 이용해서 map안에 value가 존재하면 value 값에 +1을 해주고 없을경우 1을 저장하게 됩니다.

completion는 완주한 사람을 순회해서 찾습니다.
map 안에 value값이 존재할 경우 map.set()을 통해 value값에 -1을 넣어줍니다.

이렇게 할경우 완주한 사람은 값이 0이 되고 완주를 못했을경우 1이 되게 됩니다.

이제 마지막으로 for of 문을 통해 map의 key 와 value값을 가져와서 value값이 1일 경우 key값을 꺼내주면 완주하지 못한 사람을 찾을 수 있게 됩니다.
